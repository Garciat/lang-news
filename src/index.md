# Welcome to Lang News

This is a static site built with [Lume](https://lume.land) and [Deno](https://deno.land).

## About

Lang News is your source for programming language news and updates.

## Getting Started

To run this site locally:

```bash
deno task serve
```

To build the site:

```bash
deno task build
```
